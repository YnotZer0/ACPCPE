ACPCPE Test HTML file

Test file created with Amstrad CPC

This file has been written on an Amstrad CPC6128, using Protext.

I used a mix of printer control characters (for bold, italics and underlined) and HTML tags written directly in the word processor.

In Protext is better to disable the Word-Wrap for generating HTML files.

This file is included in my Github repository for ACPCPE.

  • When entering HTML Links special attention has to be used. The entire a href must be in a whole line, otherwise ACPCPE will add a
    and will break the link.
  • A title can be created by adding the tags in the text.
  • br tags are necessary, because ACPCPE will remove the ones automatically created by the translation of this text into HTHML
